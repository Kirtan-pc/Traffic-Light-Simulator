# Traffic Light Simulator

A simple interactive web-based traffic light simulator that demonstrates the standard traffic light cycle (Red → Yellow → Green).

## Features

- **Realistic Traffic Light Display**: A styled traffic light with three lights (red, yellow, and green)
- **Automatic Cycling**: The lights automatically cycle through the standard sequence
- **Interactive Controls**: Start and stop buttons to control the simulation
- **Visual Effects**: Glowing effects on active lights for better visibility

## How to Use

1. Open `index.html` in your web browser
2. Click the **Start** button to begin the traffic light cycle
3. The lights will automatically cycle through:
   - Red (stop)
   - Yellow (caution)
   - Green (go)
4. Click the **Stop** button to pause the simulation at any time
5. Click **Start** again to resume
